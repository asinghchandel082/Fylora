# Fylora PDF Studio

Files You Love, Organized Right Away.

A modern, privacy-first PDF toolkit designed for everyday work. Built to be fast, secure, and thoughtfully simple.

## Features

- **100% Client-Side Processing**: Files never leave your browser for absolute privacy and security.
- **Premium UI/UX**: Built with React, Tailwind CSS, Framer Motion, and shadcn/ui.
- **Tools**: Merge, compress, encrypt, unlock, extract, annotate, redact, and convert PDFs.

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
